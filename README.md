# 📧 SMS Spam Detector

Classifies messages as spam or ham using TF-IDF vectorization and Multinomial Naive Bayes.

## 🚀 Live Demo
[Try it here](https://huggingface.co/spaces/Shreyas113/sms-spam-detection)

## Results
- Accuracy: ~97%
- Dataset: UCI SMS Spam Collection (Kaggle)

## Project Structure
- `sms-spam-detection.ipynb` — full notebook with EDA, training and evaluation
- `app.py` — Gradio web app for live predictions

## How to run locally
pip install -r requirements.txt
# Open sms-spam-detection.ipynb in JupyterLab

## Tech Stack
Python, scikit-learn, pandas, seaborn, matplotlib, gradio
